HOW TO INSTALL:
1. Download the game file
2. Download and install unity (2017.3.1)
3. Unzip the gamefile
4. Go into Quest/Assets/Scenes and double click "playField.unity"
5. Set the aspect ratio to 16:9
6. Press the play button


Logged Scenarios can be found at
- Quest/LogScenario
- Quest/Logs
